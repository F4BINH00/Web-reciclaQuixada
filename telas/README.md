## Exemplo

- [Tela de login](#)
  - Tela responsável por apresentar o formulário que o usuário utilizar para se autenticar na aplicação
- [Tela principal do sistema](dashboard.html)
  - Tela inicial exibida após o login do usuário. Apresenta um painel geral da aplicação, contendo menu de navegação, barra de busca, cards informativos sobre pontos de coleta, coletas disponíveis, quantidade de descartes registrados e impacto ambiental. Também possui uma seção com pontos de coleta próximos, próxima coleta agendada e materiais mais descartados.
