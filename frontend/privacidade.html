Política de Privacidade

A plataforma Recicla Quixadá coleta apenas os dados necessários para cadastro, identificação do usuário e funcionamento das solicitações de coleta.

As informações fornecidas, como nome, e-mail, telefone, endereço e documento, são utilizadas para viabilizar o contato e a organização das ações de descarte.

Os dados não devem ser compartilhados publicamente pela plataforma e devem ser utilizados apenas para os fins relacionados ao projeto.
